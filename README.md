# fourier_contour_drawer

![Fourier Contour Animation](output/nono_le_petit_robot_fourier_contour.gif)

## dependency

uv : https://github.com/astral-sh/uv

## Setup

Initialize the virtual environment with uv:

```bash
uv venv --python 3.12
source .venv/bin/activate
uv sync
```

## Run

Run the main script:

```bash
python main.py
```
